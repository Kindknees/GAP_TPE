# 考量營運限制之桃園國際機場登機門再指派研究

### A Study on Gate Reassignment at Taoyuan International Airport Considering Operational Constraints

運管系專題：針對桃園國際機場（TPE）在疫情後航空量回升所面臨的登機門調度挑戰，開發一套結合整數規劃（Integer Programming）與啟發式演算法（Tabu Search）的登機門再指派決策支援系統。

## 摘要

登機門再指派問題（Airport Gate Reassignment Problem, AGRP）是機場每日營運的核心。本研究與桃園國際機場合作，蒐集實際航班與營運限制資料，建立了一套貼近實務的數學模式，並設計高效演算法以因應即時再指派需求。

* **核心挑戰**：傳統模型多假設航班確定，忽視了航空公司偏好、機型相容性、相鄰登機門限制及 A380 特殊條件等實務限制。
* **解決方案**：
1. **精確解**：使用 Gurobi 求解整數規劃模式。
2. **啟發式解**：設計禁忌搜尋法（Tabu Search）進行大規模快速求解。


* **研究成果**：禁忌搜尋法僅需約 **17 秒** 即可產生可行解，目標式值較原始指派改善約 **33%**，遠端停機坪使用量從 57 架次降至平均 **20.74 架次**。

## 數學模型與限制式
本模型考慮了多項機場實務限制：

- 航空公司偏好：有些航空公司會有偏好的登機門。

- 機型相容性：確保登機門空間足以停靠該航班機種。

- 相鄰限制：考量相鄰登機門（Adjacent Gates）之安全間隔與干擾限制（如：A380旁邊不能停靠中型飛機，僅能停靠小型飛機）。

## 實驗結果

| 指標 | 桃園機場原指派 | 數學模式 (Gurobi) | 禁忌搜尋法 (TS) |
| --- | --- | --- | --- |
| 求解時間 (s) | - | 90 ~ 2400 | **~17.36** |
| 目標式值 | 5097 | 2065 ~ 2940 | 3411.98 |
| 派至停機坪次數 | 57 | 23 ~ 27 | **20.74** |

研究證實，禁忌搜尋法在求解品質與計算效率間達到良好平衡，適合機場營運單位的決策支援需求。
